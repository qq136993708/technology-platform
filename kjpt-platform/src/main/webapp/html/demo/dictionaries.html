页面多标签动态赋值
这里是动态赋值:
这里的值自动从字典里获取：
从字典里去多值：

<div class="text-red" diy-form-value="name"></div>
<div class="text-red" diy-form-value="value" diy-dic-data="DIC_DEMO_DATA"></div>
<div class="text-red" diy-form-value="moreValue" diy-dic-data="DIC_DEMO_DATA"></div>
diy-form-value="name" => 必填；且值不能为空
diy-dic-data="DIC_DEMO_DATA" => 如果需要从字典中取值时，需要添加字典ID

var targetData = { name: '这里是动态赋值', value: '2', moreValue: '2,3,4'};
// 调用 common.js 里的公共方法，并把数据传入即可; 碰到字典时会自动加载并筛选数据
// 从字典里去多值时用 逗号（,）隔开
setTargetNameValue(targetData);
